# Database Web Viewer

**Description:**

Using Jsgrid to dispay table informations.

1. Support the basic filter.
2. Support MongoDB

**Usage:**

1. Starting the web server

   go run myweb.go
   
2. View the table information using any web browser, e.g.

   http://127.0.0.1/db/TestDB/Table1
